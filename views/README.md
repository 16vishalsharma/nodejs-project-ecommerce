# Views Directory

This directory is for storing view-related files. You can use this for:

1. **API Documentation Views** - HTML/JSON views for API documentation
2. **Email Templates** - If you need to send emails
3. **PDF Templates** - For generating invoices, receipts, etc.
4. **Frontend Views** - If you're building a server-side rendered application

## Structure Suggestions

```
views/
  ├── emails/
  │   ├── welcome.html
  │   ├── order-confirmation.html
  │   └── password-reset.html
  ├── templates/
  │   ├── invoice.html
  │   └── receipt.html
  └── api-docs/
      └── index.html
```
